1. Original Hypothesis
Pre-event intraday pressure contains separable:
Information-driven component
Liquidity-driven component
Info component should amplify PEAD.
2. Implementation
Built TAQ pipeline
Extracted ~20 intraday features
Constructed 4 composite pressure measures
Tested via quintile backtests and clustered regressions
3. Empirical Findings
Strong short-term reversal pattern
Weak or non-monotonic medium-term drift
PCA indicates dominant liquidity factor
Composite measures heavily correlate with size/liquidity state
No stable evidence of clean information-driven drift
4. Diagnosis
Current features are structural mixtures of info and liquidity
Cross-sectional normalization introduces liquidity/size contamination
No feature behaves like a clean permanent-impact proxy
Information/liquidity separation is fundamentally latent
5. Next Step
Move toward theory-anchored measures:
Kyle’s λ
Permanent vs transitory impact decomposition
Orthogonalized pressure vs liquidity state
Possibly VPIN-style imbalance intensity
